# RutorPreviewAjax

![License](https://img.shields.io/github/license/alekpet/Rutor-Preview-Ajax)

Расширение для удобного просмотра раздач на сайте **Rutor** без необходимости открывать каждую страницу отдельно.

## Возможности

- 📂 Быстрый предпросмотр раздач прямо на странице поиска или каталога.
- 💬 Комментарии автоматически скрываются в спойлер для экономии места.
- 📑 Панель открытых раздач с возможностью быстро переключаться между ними.
- 🔽 Кнопка для сворачивания всех открытых предпросмотров.
- ⭐ Добавление раздач в избранное.
- ✅ Проверка актуальности раздач, сохранённых в избранном.
- 📊 Сортировка раздач по различным параметрам:
  - дате;
  - названию;
  - размеру;
  - количеству раздающих (Seed);
  - количеству скачивающих (Peer).
- ▶️ Воспроизведение видео и аудио прямо в браузере через:
  - [TorrServer](https://github.com/YouROK/TorrServer/releases) *(рекомендуется)*;
  - [Ace Stream](https://www.acestream.org/?lang=ru).

## Преимущества

- Не нужно открывать каждую раздачу в новой вкладке.
- Экономия времени при поиске нужного контента.
- Более удобная навигация по большому количеству раздач.
- Возможность сразу просматривать описание, комментарии и запускать воспроизведение.

## Установка

1. Установите один из менеджеров пользовательских скриптов:
   - Tampermonkey *(рекомендуется)*;
   - Violentmonkey;
   - Greasemonkey.
2. Установите скрипт **RutorPreviewAjax.user.js**.
3. Откройте сайт Rutor - скрипт активируется автоматически.
